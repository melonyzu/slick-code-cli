# Support

Thanks for using Slick Code CLI. Here is where to go depending on what you
need.

## Questions and usage help

- Read the [README](README.md) for installation and everyday usage.
- Run `/help` inside the interactive assistant for the list of slash
  commands, and `slickcode --help` for the command tree.
- Ask usage questions on [GitHub Discussions][discussions].

## Bugs

Open a [bug report][issues] with reproduction steps, the command you ran,
and the output you saw. `slickcode version` prints the build information we
need.

## Feature requests

Open a [feature request][issues] describing the problem you are trying to
solve. Explaining the motivation helps us find the right design.

## Security issues

Do not open a public issue for security reports. Follow the process in
[SECURITY.md](SECURITY.md).
